It is reccomended to install these dependencies in a virtual environment, this is facilitated by {\ttfamily python}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ python3\ -\/m\ venv\ venv}
\DoxyCodeLine{\$\ source\ venv/bin/activate}

\end{DoxyCode}


Packages can thus be installed using {\ttfamily pip install \texorpdfstring{$<$}{<}package\texorpdfstring{$>$}{>}}.

Install the following requirements for instantiating and running the model.


\begin{DoxyCode}{0}
\DoxyCodeLine{keras==2.4.1}
\DoxyCodeLine{tensorflow==2.4.1}
\DoxyCodeLine{opencv-\/python==4.6.0}

\end{DoxyCode}


Install these requirements for model training


\begin{DoxyCode}{0}
\DoxyCodeLine{imgaug==0.4.0}
\DoxyCodeLine{numpy==1.23.5}

\end{DoxyCode}


Alternatively, to install all the required packages, use our requirements file\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{pip\ install\ -\/r\ requirements.txt}

\end{DoxyCode}


To use the file {\ttfamily \doxylink{tflite__model__runner_8py}{tflite\+\_\+model\+\_\+runner.\+py}}, you must install the tflite runtime, this can be useful on devices where you just want to execute the model and don\textquotesingle{}t nessicarily want to install the whole tensorflow package.


\begin{DoxyCode}{0}
\DoxyCodeLine{pip\ install\ tflite\_runtime}

\end{DoxyCode}
