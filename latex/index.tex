This repository contains accompanying code, documentation and guides supporting our paper\+: \href{https://arxiv.org/abs/2411.14467}{\texttt{ Towards Scalable Insect Monitoring\+: Ultra-\/\+Lightweight CNNs as On-\/\+Device Triggers for Insect Camera Traps}}. Ecto-\/\+Trigger is software to produce lightweight binary classification models to detect images containing objects of interest, this is intended to be a lightweight alternative to object detection models, for scenarios where the location of the object is not required but computationally efficient models are important.

Ecto-\/\+Trigger has been developed for insect camera traps, where insects (ectotherms) cannot activate traditional PIR camera trap triggers. Lightweight binary classifiers can be deployed locally on microcontrollers and continuously run to filter insect images in real-\/time.

This repository contains information to build, deploy and evaluate your own binary classifiers, as well as the existing model weights from our paper for recognising images containing insects.\hypertarget{index_autotoc_md15}{}\doxysection{\texorpdfstring{Guides}{Guides}}\label{index_autotoc_md15}
